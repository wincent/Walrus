Purchasing Hextrapolate

Purchase Hextrapolate and you will receive a license code via email as soon as the transaction is complete.

To purchase a Hextrapolate license:

  1. From the Hextrapolate menu select “Buy now…”
  2. Choose the appropriate license type (single-user, multi-user) and proceed to the checkout
  3. An automated system will issue you with a license code via email

Once you have received your license code you may use it to register Hextrapolate.
