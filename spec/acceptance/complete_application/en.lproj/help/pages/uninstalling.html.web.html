Uninstalling Hextrapolate

Uninstalling Hextrapolate is a simple two-step process:

  1. Using the Finder navigate to the folder where you have Hextrapolate installed (most commonly the Applications folder).
  2. Drag the Hextrapolate application icon to the Trash.
