Transferring your license to another machine

In order to transfer your Hextrapolate license from one machine to another you first unregister the existing installed copy and then register the copy on the new machine.

You can do this from within the Hextrapolate registration preference pane. Alternatively, you can copy the license file from one machine to another as described below.

To manually copy your license from one machine to another:

  1. Quit Hextrapolate.
  2. Using the Finder, go to your home directory and open the Library folder
  3. Inside the Library folder, open the Preferences folder
  4. Copy the file com.wincent.Hextrapolate.license.plist from the Preferences folder to the corresponding folder on your target machine.

There is no need to copy your activation certificate (com.wincent.Hextrapolate.activation.certificate.plist) to the new machine as these certificates are specific to the machine on which you performed the activation.
