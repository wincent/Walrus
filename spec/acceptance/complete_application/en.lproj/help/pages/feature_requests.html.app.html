How to submit a feature request

There are two avenues for submitting a feature request for Hextrapolate:

  • Submit a feature request via the web at bugs.wincent.com
  • Submit a feature request via email to features@wincent.com

Submitting a feature request via the web is strongly recommended because you can use the web interface to perform a search before submitting (it may be that someone else has already requested the enhancement).
