Obtaining a System Profiler report

To provide detailed troubleshooting information with your bug report you can attach a System Profiler report.

To obtain a System Profiler report:

  1. Open the System Profiler application (inside the Utilities folder in the Applications folder).
  2. From the File menu, choose Export.

System Profile reports can be quite large so you may want to compress it prior to attaching it to your bug report. One way of doing this is to click on the report in the Finder while holding down the Control key, then select “Create Archive” from the contextual menu.
