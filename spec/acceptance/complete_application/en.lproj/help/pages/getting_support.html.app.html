Requesting support

Before submitting a support request please try the troubleshooting tips. Consider requesting a feature, reporting a bug or sending feedback rather than sending a support request if it seems more appropriate.

To request Hextrapolate support:

  1. Perform basic troubleshooting as described in troubleshooting tips.
  2. If you think you have discovered a bug, consider submitting a bug report.
  3. Contact me via email to support@wincent.com.
