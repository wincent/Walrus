Installing Hextrapolate

Hextrapolate uses a “drag-and-drop” installation procedure. This means that you can install it wherever you wish. The most common location is the Applications folder.

Before installing you will need a copy of the Hextrapolate disk image (see downloading Hextrapolate).

To install Hextrapolate:

  1. In the Finder, double-click on the Hextrapolate disk image to mount it.
  2. Once the disk image has mounted a window will appear in which you can see the Hextrapolate application.
  3. Drag the Hextrapolate application to a location of your choosing.

If you wish to try Hextrapolate without installing it you can run it from the mounted disk image by double-clicking the Hextrapolate application icon. If you do this you will need to quit Hextrapolate before unmounting the disk image.
